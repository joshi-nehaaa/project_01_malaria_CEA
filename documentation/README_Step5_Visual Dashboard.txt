Step 5: Visual Dashboard

Objective:
	Create a concise one-pager dashboard summarising the cost-effectiveness results of the ITN distribution programs in Nigeria, 2021
	
Data Source: 
	Sheet impact_visuals_2021: charts
	Sheets impact_model_2021 & zone_metrics_2021: KPIs & insights
	
Steps: 
	1. Create a new sheet within impact_model_2021 file titled: dashboard_2021
	2. Define a layout for:
		Header: project title
		KPI cards: 5 key indicators of the programs
		Charts: visuals showing cost-effectiveness & behavioural linkages
		Insights: qualitative interpretation & implications
	3. Reuse visuals from previously created sheet impact_visuals_2021
	4. Extract and incorporate KPIs & insights from the analysis of the previous sheets: impact_model_2021, zone_metrics_2021
	5. Export as a single PDF: dashboard_2021

Key Findings:
	Average Cost per DALY Averted: $3098
	Average Cost per Case Averted: $62		
	Total DALYs Averted: ≈27,520		
	Most Cost-Effective Zone: North West	
	Average Usage Gap: ≈17.6%
